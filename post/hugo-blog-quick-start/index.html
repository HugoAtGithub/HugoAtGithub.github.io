使用 Hugo 快速搭建个人博客

1. Hugo 安装
  • 官方安装教程

这里以 Windows 系统为例：

  1. 到 Hugo 的 GitHub 页下载 hugo_extended_xxx_Windows-64bit.zip;

  2. 解压到一个目录，如： C:\hugo

  3. 将文件目录 C:\hugo 添加到 环境变量 PATH

  4. 重启 cmd 终端，运行 hugo version ，如下提示说明安装成功：

2. 创建一个 Site

这里的 Site 可以说是一个 hugo 博客项目， 用于生成最终的博客网站代码

官方快速开始教程

  1. 进入到一个目录，用于生成site文件， 如： C:\hugo\sites

  2. 执行命令

    hugo new site quickstart

  3. 上一个命令就是在当前目录的 quickstart 文件夹下生成一个 hugo 博客项目

3. 添加主题

​ 这里用 Ananke 主题， 其他主题可以到 themes.gohugo.io 挑选。

  1. 先从 github 下载一个主题到 themes 目录

    cd quickstart
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

  2. 添加主题到 site 的配置文件：

    echo 'theme = "ananke"' >> config.toml
4. 添加博客内容

  1. 使用 new 命令添加一个文章

    hugo new posts/my-first-post.md

  2. 编辑文章

---
title: "My First Post"  # 这里是文章标题
date: 2019-03-26T08:47:11+01:00
draft: true  # 草稿模式
---
这里编辑文章内容
5.启动一个 Hugo 服务
  1. 执行
hugo server
  1. 浏览器打开 http://localhost:1313/，即可看到
6.生成静态页面

直接执行

hugo

就会生成一个 ./public 目录，目录下的文件就是我们博客网站用到的所有文件

7.上传网站到 Github

现在我们就是要将我们刚生成的网站静态页面文件上传到 github，并利用 GitHub Pages 浏览我们的网站。

这样我们即使没有服务器和域名，也能搭建我们自己的网站，并让别人可以浏览。

  1. 首先创建一个 GitHub 仓库

    1. 仓库名为 你的 GitHub 用户名.github.io， 比如我的就是 hugoatgithub.github.io

    2. 然后直接创建仓库

    3. 最后就是根据 GitHub 提示，将本地 public 的代码上传到这个项目就可以了
